int  led = -1,
uint32_t  memOffset = 0

Checks if fail safe mode should be activated. This should be called at the beginning of main setup function.

Parameters
maxBootCyclesNumber of cycles after device enters in Fail Safe mode
ledIndicates Fail Safe mode status. Normally off (= HIGH)
memOffsetRTC memory offset. This is not used if storage is SPIFFS
Parameters
maxBootCyclesNumber of quick boot loops on which fail safe mode is started
memOffsetIf RTC is used this is offset. Modify it if your project uses RTC memory too
